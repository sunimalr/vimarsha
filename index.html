vimarsha

Performance Analysis using Machine Learning

What is vimarsha?

vimarsha is a collection of tools that can be used to identify memory related issues in a parallel program. Data collection tool in the tool package is used to collect performance related event counts for a program. Then using the data visualization tool, a user can get information about memory related issues in the program.

Overview

Computer hardware is being developed at an increasingly rapid pace. Denser, faster and complex processors are introduced into the market to meet the need of computational tasks which demand high performance. Mostly due to the physical limitations in the enhancement of processors, multi-core processing systems have become the viable path to follow to gain more performance. Additionally, more sophisticated memory schemes (different levels of caches, cache coherency protocols, buses, etc) have been introduced to improve the performance of the multi-core processors.

With such advancements in multi core processor technologies, programmers have been able to improve software performance significantly by using parallelization of programs/execution, expecting to distribute the processing needs over multiple cores. However, sometimes the expected performance boost cannot be harnessed in parallel programs due to various causes. In some cases, the performance of parallelized version of a program is even worse than its sequential version. When it comes to high performance computing, such performance flaws can be a serious problem. Yet, finding out the causes of performance problems can itself be difficult.

Vimarsha uses machine learning techniques to identify memory related issues that affect multi-threaded programs running on multicore processors. This will help developers to identify memory related issues in programs.
